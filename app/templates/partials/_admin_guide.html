{# 1. Book cycle #}

Running a book cycle

  1. A member submits a suggestion. Review it under Pending suggestions — approve to add to the queue, or reject it.
  2. When ready, click Set as current on any queued book. Fill in the first reading assignment and total chapters.
  3. Each meeting, click Edit details on the current book to advance the reading assignment to the next chapters.
  4. When finished, click Mark completed. This moves it to past books and auto-completes all member progress records.
{# 2. Reading assignment #}

Setting the reading assignment

  • The From and To chapter fields define what members should read before the next meeting.
  • Single chapter: enter the same number in both fields — e.g. From 2 To 2. The To field auto-fills when you tab away from From.
  • Range: enter different numbers — e.g. From 1 To 3. Members see "Chapters 1–3".

A live preview appears as you type so you can see exactly what members will read.

{# 3. Access code #}

Access code

  • The access code gates entry to the platform. Only people with the code can reach the dashboard.
  • Change it when you want to refresh membership access. Use Generate for a random code, or type your own.
  • Existing members won't be locked out — they're already logged in. New members need the new code to register.

Always post the new code in the WhatsApp group immediately after changing it so new members can join.

{# 4. Meetings #}

Meetings

  • There is one meeting record that you update each cycle. It shows as a banner on the member dashboard.
  • Date and time are entered in UK time (Europe/London) and automatically converted for the system.
  • Tick Cancel this meeting to show a cancellation banner instead. Add an optional note — e.g. "Taking a break for the holidays".
  • Members see a Join Now button when the meeting is within 30 minutes of starting.
{# 5. Members #}

Managing members

Promoting to admin

Grants full access to this admin panel. Only do this for trusted members — you can demote them later.

Last admin protection

You can't demote yourself if you're the only admin, and you can't delete your account either. Promote someone else first.

Good habit — always keep at least two admins active so the club isn't blocked if one account becomes inaccessible.
